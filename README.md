# Forecasting in the Italian day-ahead electricity market

This repository contains the code used for the empirical analysis of the Master's Thesis *Forecasting in the Italian day-ahead electricity market*.

The project studies day-ahead hourly forecasting in the Italian electricity market, focusing on the seven physical bidding zones of the Mercato del Giorno Prima (MGP). The forecasting targets are zonal electricity prices, accepted purchase volumes and accepted sale volumes.
